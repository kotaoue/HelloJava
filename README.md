# HelloJava
Java in the morning is delicious.

## Requirements

- macOS
- Java Development Kit (JDK)

## Installation

Install JDK using [Homebrew](https://brew.sh/):

```sh
brew install openjdk
```

After installation, follow the instructions shown by Homebrew to add Java to your PATH. For example:

```sh
echo 'export PATH="/opt/homebrew/opt/openjdk/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc
```

Verify the installation:

```sh
java -version
javac -version
```
